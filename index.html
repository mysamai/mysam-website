Sam is an open-source,

web-based "intelligent" assistant.

npm install mysam -g
Hi! I'm Sam. What's your name?

Sam can listen to you, learn new actions and is extensible with custom plugins.

Using machine learning to process your voice and chat messages, Sam gets smarter as you use and teach it.

Built entirely in JavaScript, Sam runs in any modern webbrowser. No server or third party services required.
