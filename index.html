Sam is an open-source,

web-based "intelligent" assistant.

npm install mysam -g
Hi! I'm Sam. What's your name?

Sam can listen to you, learn new actions and is extensible with JavaScript plugins.

Sam runs a NodeJS server and in any modern browser

Sam runs can also run as an Electron desktop application.
